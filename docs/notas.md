# Notas de la práctica 
- Aprendí a crear carpetas en visual estudio code. 
- Aprendí a crear archivos dentro de un repositorio.

Cuestionario: 

- ¿Qué fue lo que más te gustó de trabajar con GitHub Desktop?
Lo que mas me gusto fue que los archivos y las versiones de estos, se suben a la carpeta. 
- ¿Qué te pareció más difícil?
Me parecio mas dificil la creación de la cuenta, por el momento los demas pasos son muy faciles. 
- ¿Para qué crees que sirve un archivo .gitignore?
El archivo .gitignore sirve para no subir documentos deseados de una versión, por ejemplo las notas que se van tomando de un proyecto y se guardan como un archivo en VS code. 
- ¿Por qué es importante escribir buenos mensajes en los commits?
Es importante escribir buenos mensajes en los commits porque así tenemos un mayor entendimiento de la versión que se ha subido.
